# Contributing

Home Cinema Control is currently source-available, not open source.

The repository is public so users can inspect the code, run HCC for personal
non-commercial use, report problems, and understand how the application works.
External code contributions are not accepted automatically.

Before submitting a pull request with code, documentation, screenshots, assets,
or translations, open an issue first and wait for maintainer confirmation.

Unless a separate written agreement says otherwise, by submitting a contribution
you confirm that:

- you have the right to submit it;
- it is your original work or is compatible with HCC's licensing model;
- you grant the project maintainer a perpetual, worldwide, irrevocable,
  royalty-free right to use, modify, distribute, sublicense, and relicense it as
  part of Home Cinema Control.

Do not submit copied code, screenshots, documentation, or assets from projects,
forums, vendors, firmware, or third-party websites unless the rights are clear
and compatible with HCC's licensing model.
